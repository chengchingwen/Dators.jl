# Dators.jl

A data implementation.

```julia
julia> using Dators

julia> d = Dator(i->i*i, 1:100; mode=Dators.Thread(2))
Dator{Any, Int64, Dators.Thread(2)}(Init)

julia> d = Dators.batch(d, 5)
Dator{Any, Any, Dators.Thread(1)}(Init)
```
